# Contributing Guidelines

Thank you for your interest in contributing to Awesome GEO! 🎉

## 📋 Contribution Process

### 1. Submitting New Resources

If you want to add a new resource, please ensure:

- ✅ The resource is directly related to GEO (Generative Engine Optimization)
- ✅ The resource is high-quality and valuable to the community
- ✅ The resource link is valid
- ✅ The resource is not a duplicate

### 2. Format Requirements

#### Resource Format

```markdown
- [Resource Name](link) - Short description (no more than one line)
```

#### Table Format

```markdown
| Tool Name | Description | Link |
|-----------|-------------|------|
| **Tool** | Short description | [Link text](URL) |
```

### 3. Submission Steps

1. **Fork the Repository**
   ```bash
   git clone https://github.com/luka2chat/awesome-geo.git
   cd awesome-geo
   ```

2. **Create a Branch**
   ```bash
   git checkout -b add-new-resource
   ```

3. **Add Your Resource**
   - Add your resource under the appropriate category
   - Sort alphabetically
   - Ensure consistent formatting

4. **Commit Your Changes**
   ```bash
   git add .
   git commit -m "Add [Resource Name] to [Category Name]"
   ```

5. **Push and Create PR**
   ```bash
   git push origin add-new-resource
   ```
   Then create a Pull Request on GitHub.

## 📝 Pull Request Guidelines

### PR Title Format

- `Add [Resource Name]` - Adding new resource
- `Update [Resource Name]` - Updating existing resource
- `Remove [Resource Name]` - Removing invalid resource
- `Fix [Issue Description]` - Fixing issues

### PR Description Template

```markdown
## Change Type
- [ ] Adding new resource
- [ ] Updating existing resource
- [ ] Removing invalid resource
- [ ] Fixing errors
- [ ] Other

## Resource Information
- **Name**: 
- **Link**: 
- **Category**: 
- **Description**: 

## Checklist
- [ ] I have confirmed the link is valid
- [ ] I have checked the resource is not a duplicate
- [ ] I have followed the format guidelines
- [ ] The resource is related to GEO
```

## 🏷️ Resource Categories

Please add resources to the correct category:

| Category | Description |
|----------|-------------|
| Learning Resources | Papers, articles, tutorials, videos, podcasts |
| Tools & Platforms | AI monitoring, content optimization, brand monitoring tools |
| AI Search Engines | Conversational AI, AI-enhanced search, domain-specific search |
| Strategies & Practices | Content strategy, technical optimization, authority building |
| Analytics & Monitoring | Metrics, workflows, monitoring methods |
| Case Studies | Success stories, industry analysis |
| Communities & Forums | Discussion communities, forums |
| News & Trends | Industry blogs, newsletters |
| Books | Recommended books |

## ⚠️ Important Notes

### Resources We Accept

- ✅ Free tools or tools with free versions
- ✅ High-quality learning resources
- ✅ Actively maintained projects
- ✅ Content with practical value

### Resources We Don't Accept

- ❌ Pure advertising or marketing content
- ❌ Low-quality or outdated resources
- ❌ Content unrelated to GEO
- ❌ Paywalled content (unless exceptionally valuable)
- ❌ Duplicate resources

## 🤝 Code of Conduct

- Respect all contributors
- Maintain friendly and constructive discussions
- Accept constructive feedback
- Focus on what's best for the community

## 💡 Suggesting New Categories

If you think a new category is needed:

1. Discuss it in Issues first
2. Explain why this category is needed
3. Provide at least 3 resources that fit the category
